Hi Starfam,

We’ve just completed the 15-day Starcoin 2.0 trial run points challenge, and we’re very grateful for your participation. This event not only provided great help in improving the performance and stability of the main network, but also marked the beginning of a new stage for Starcoin. Here are some exciting metrics:

### Excellent Technical Performance

The Starcoin 2.0 mainnet has significantly improved network performance by introducing FlexiDAG and TurboSTM parallelization technologies. In the trial environment:
- **Maximum Block Generation Speed (BPS):** Reached 1 block/s.
- **Maximum Parallelism:** Achieved 2 blocks concurrently.
- **Maximum Transaction Processing Speed (TPS):** Reached 130k tx/s.

This excellent performance surpasses more than 90% of the public chains, and at the same time strengthens our confidence in Starcoin becoming the backbone of the Web3 network infrastructure.

### Mining Points Rewards

This Starcoin 2.0 Mining Points Challenge event allowed more users to spearhead the excellent performance of the 2.0 mainnet and provided valuable data support to the development team. The event has been a resounding success! For users who participated, you can view the points reward details through the link below:

### Join the Starcoin 2.0 Future

We sincerely invite you to join Starcoin 2.0, become part of our global node, and participate in the construction and ecological development of the Starcoin scalability network. You can follow the instructions below:

**How to Participate?**
1. **Node Download:** [Download Starcoin 2.0 Node](https://github.com/starcoinorg/starcoin/releases/tag/v2.0.0)
2. **Mainnet Startup Parameters:** [Vega Network Setup Instructions](https://cookbook.starcoin.org/docs/getting-started/setup/vega-network/)
3. **Node Verification:** Verify transactions by running a node. Nodes that successfully participate in the proof of computing power will receive new block rewards.
4. **Wallet and Transactions:** Connect to the 2.0 network using StarMask to send and submit transactions.
5. **Applications:** Developers can deploy Move applications on the 2.0 network and experience its powerful features.
6. **Incentives:** The trial run phase is intended to verify the robust features of 2.0. Note that there is no specific economic model during this phase, and it will not replace the current 1.0 version for the time being.

To connect your StarMask wallet to the 2.0 network, use the following configuration:
- **Network Name:** vega
- **RPC URL:** https://vega-seed.starcoin.org
- **Chain ID:** 2
- **Currency Symbol:** STC
